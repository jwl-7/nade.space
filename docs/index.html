showMap

Displays the map page.

showMap(mapName: string, callback: callback): boolean
Parameters
mapName (string) The filename of the map's JSON file.
callback (callback) Callback function useful for updating the map page.
Returns
boolean: True - by default, False - if mapName can't be found.

showImage

Displays the nade image/gfy page.

showImage(mapName: string, team: string, nadeType: string, id: integer): boolean
Parameters
mapName (string) The filename of the map's JSON file.
team (string) The team that the nade is designated for - CT(Counter-Terrorists) / T(Terrorists).
nadeType (string) The type of nade being thrown - Smokes, Fires, Flashes, Wallbangs.
id (integer) The number of the nade - identifies which nade to show.
Returns
boolean: True - by default, False - if mapName, nadeType, or id can't be found.
