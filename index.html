About

nade.space is a minimalistic reference tool, designed with the aim of giving players a simple resource for studying practical smokes, flashes, fires, and wallbangs. This website is NOT a complete nade database.
simple. minimal. practical.

nade.space in 7 easy steps:
  1. Run practice config in offline server.
  2. Open nade.space in the Steam browser.
  3. Select a map.
  4. Select a nade.
  5. Click copy location.
  6. Paste location into console.
  7. Throw nade.

Definitions and References
  • Item Reference
    •   Smokes: Smoke Grenades
    •   Flashes: Flashbang Grenades
    •   Fires: Incendiary Grenades / Molotovs
    •   Wbangs: Wallbangs
  • Nade Info Format
    • From: [start location]
    • To: [end location]
    • Side: [team]
    • Type: [nade instructions]
    • Additional details / instructions are specified in brackets '[]'
  • Throw Types
    • Throw: left-click
    • Toss: right + left-click
    • Lob: right-click
    • Jump-Throw: jump and throw simultaneously
    • Run-Throw: run forward a few steps, then throw
    • Step-Throw: run forward 1 or 2 steps, then throw
    • Walk-Throw: walk forward 1 or 2 steps, then throw
    • Crouch-Throw: crouch, then throw
    • Break-Throw: break glass, then throw
  • Wbang Types
    • Solid: only viable with sniper rifles [scout, awp, auto]
    • Thin: viable with assault rifles [ak47, m4a4, m4a1]
  • Detail Types
    • [A / B]: A Site / B Site
    • One-Way: self-referenced smoke location used for an unfair advantage
    • Apts: apartments
    • Cross: exposed area that Terrorists use to take site
    • Safe: the 'default' plant position where Terrorist can safely plant bomb
    • [▲▼◄►]: represents specific direction of location
